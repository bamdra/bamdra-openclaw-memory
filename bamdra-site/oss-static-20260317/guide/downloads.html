Recommended path ​

  1. If supported, install with openclaw plugins install @bamdra/bamdra-openclaw-memory.
  2. Otherwise download the release package and follow the manual install guide.
  3. Let the main plugin auto-provision bamdra-user-bind and auto-create the memory directory on first bootstrap.
  4. Merge the config sample into your OpenClaw setup.
  5. Track new releases from GitHub or npm.
  6. Add bamdra-memory-vector only when you want semantic retrieval enhancement.
